# E-Commerce Sales Analysis Using SQL

## Project Objective
Analyze e-commerce sales data using PostgreSQL to identify revenue, profit, customer behavior, and category performance.

## Tools Used
- PostgreSQL
- pgAdmin 4
- GitHub

## Business Questions

1. What is the total revenue, total profit, and total quantity sold?
2. How much sales and profit has each category generated?
3. Which are the top 3 states with the highest sales?
4. Which customer segment generates the highest profit?
5. What is the average sales value and average profit per order?
6. Which sub-categories are generating losses?
7. How many unique customers have placed orders?
8. Which orders generated sales above 1000 but resulted in a loss?
9. How many orders were placed in each city?
10. Which Furniture sub-category has the highest sales?
11. Which categories have sales above average sales? (Subquery)
12. Classify orders into High, Medium, and Low Sales using CASE WHEN.

## Key Insights
- Total Revenue: 11,425
- Total Profit: 1,103
- Total Products Sold: 33
